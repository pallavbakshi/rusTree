Expand description

Sorts a vector of NodeInfo while preserving the tree structure.

This function takes a flat vector of nodes in DFS order, builds a tree structure, sorts siblings at each level according to the specified criteria, and then flattens the tree back to a DFS-ordered vector.

ยงArguments

  • nodes - A mutable reference to a vector of NodeInfo to be sorted. The initial vector is assumed to be in DFS order (e.g., from walk_directory).
  • key - The SortKey specifying the attribute to sort siblings by.
  • reverse - A boolean indicating whether to reverse the sort order for siblings.
