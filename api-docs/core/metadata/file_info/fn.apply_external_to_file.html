pub fn apply_external_to_file(
    file_path: &Path,
    ext_func: &ExternalFunction,

Expand description

Applies an external command to the file and returns its stdout as string. The command template may contain the placeholder {} which will be replaced with the file path. The implementation is best-effort and synchronous; the timeout is enforced by killing the child process if it exceeds the given duration.
